Webpages

The webpage for my Master’s thesis was designed with responsive web design. Moreover, the website was accessible to visually impaired people through the use of screen readers. My current website is responsive to different output devices.

Responsive Webdesign

"The mobile web is growing 8 times faster than the desktop web. The average time spend per day on a smartphone is more than 30 minutes."
These numbers show that responsive websites should be one of the biggest concerns for the next years of website development. With responsive web design pages adapt seamlessly to the output device.

My current portfolio website is responsive to desktop, tablet and mobile. Besides HMTL, CSS and Javascript (jQuery), it also uses Google Analytics.

Accessibility

A particular focus for me as a designer is that webpages should be accessible to the widest audience, in particular to visually impaired people. First time I came across this topic was during my study as part of project. Since then, I created several webpages according to the WAI standard (Web Accessibility Initiative). This concept allows visually impaired people to have access via a screen reader or a braille terminal. Technically, objects such as images or videos are augmented with specific tags in order to facilitate the navigation with these alternative devices.

Book and Paper Publication

RFID, Ubiquitous Computing and Responsive Web Design

My Master’s thesis focussed on fields of ubiquitous application for RFID. My theoretical framework of Ubiquitous Computing and its implementation resulted in a book publication and a peer-reviewed journal publication.

My thesis' practical implementation topic included the so-called 'Talking Points' prototype collecting information (e.g. at a fair) with a mobile device exploiting RFID technology. This information is presented through a webpage that is designed to use one single point of content, implemented with HMTL and PHP. Responsive designs enables the same page being showed on both mobile devices and standard computers.

My theoretical framework of Ubiquitous Computing and its implementation resulted in a book publication distributed over Amazon and a peer-reviewed journal publication.

In my thesis, I was able to combine many fields that I am enthusiastic about: new technologies and user interfaces, such as webpages using accessibility and responsive web design. I presented the results at a student’s conference poster session.
